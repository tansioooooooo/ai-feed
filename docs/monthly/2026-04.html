📋 月次サマリー

2026年3月は Claude Code の組織的普及が急加速し、日本企業を含む多数の組織が全社導入を進めた1ヶ月だった。仕様駆動開発・ハーネスエンジニアリングなど新たな開発パラダイムの議論が活発化した一方、LiteLLM・axiosなどサプライチェーン攻撃が相次ぎ AI 時代のセキュリティリスクが顕在化。Atlassian・Oracle・Meta など大企業がAI活用を理由に大規模レイオフを実施し、雇用への影響も現実化してきた。オープンソース LLM の急進化も続いており、Qwen 3.5・Rakuten AI 3.0 など競争が激しさを増している。

🔥 主要トレンド

Claude Code の爆発的普及と開発パラダイムシフト
個人・チーム・全社導入の事例が日本でも急増。Skills（エージェントチーム）、CLAUDE.md 設計、ハーネスエンジニアリングといった新概念が定着し始め、非エンジニアによる業務自動化事例も多数報告された。「60歳でも情熱を取り戻した」「月商300万のSaaS全コードを書かせた」など個人の体験談も多く話題に。
AIコーディングエージェント競争の激化と機能拡充
OpenCode（OSS）、Cursor Composer 2（Kimi K2.5ベース）、DeNA の Devin 全社2000人展開など競争が激化。OpenAI Codex がサブエージェント機能を追加し、Astral（Ruff/uv）を買収して Python 生態系を取り込む動きも。Opus 4.6/Sonnet 4.6 の 1M コンテキストウィンドウが一般提供開始。
オープンソース LLM の急速な高性能化
Qwen 3.5 が小規模モデル（9B）でも大規模モデルを凌ぐ性能を示し注目を集めた。楽天 AI 3.0、PLaMo 3.0 Prime など国産 LLM も登場。Flash-MoE（397B をノート PC で動作）、TurboQuant（メモリ 1/6・速度 8 倍）など推論効率化技術の進展も目立った。
AI時代のセキュリティリスクの顕在化
LiteLLM（月間 9500万 DL）と axios の npm サプライチェーン攻撃、GlassWorm による不可視 Unicode リポジトリ汚染が相次いで発覚。GitHub Copilot が PR に広告を挿入する問題（後に撤回）も起き、AIコーディングに起因する「verification debt（検証負債）」の議論も活発化した。
雇用・産業構造への影響の現実化
Atlassian 1600人・Oracle 3万人・Meta 最大20%削減計画・Epic Games 1000人超など、AI活用を背景とした大規模レイオフが相次いだ。一方「AI は作業量を増やすだけ（Amazon 社員の声）」「ホワイトカラー業務を代替しない」という反論も存在し、Anthropic による労働市場インパクト測定の研究も発表された。
AIガバナンス・法的問題の複雑化
米国防総省による Anthropic のサプライチェーンリスク指定（裁判所が後に差し止め）、AI 顔認識による冤罪事件（テネシー州女性ら）、EU の Chat Control 廃止決定など、法・倫理・安全保障面での動きが加速。著作権訴訟の決着や RAG へのドキュメント汚染攻撃なども議論された。
