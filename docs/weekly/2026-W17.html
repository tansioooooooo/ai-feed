📋 週次サマリー

2026年4月9〜15日の一週間、Claude Code の企業展開と Google Gemma 4 のリリースが最大の話題となった。コーディング未経験者の86%がデプロイ達成という事例や Anthropic によるデスクトップ版刷新（マルチエージェントUI対応）が注目を集めた。一方、Gemma 4 の iPhone ネイティブ動作によるローカルAI推論の実用化も加速。CPU-Z/HWMonitor へのマルウェア混入、WordPress プラグインへのバックドアなどセキュリティ事件も頻発し、AIの社会・経済的影響への警戒感も高まっている。

🔥 主要トレンド

Claude Code の企業・組織展開が急加速
グッドパッチがコーディング未経験者86%デプロイ達成を公表、メルカリがセキュリティ設定の組織配布戦略を公開、Anthropic がマルチエージェントUIに対応したデスクトップ版を刷新。企業レベルの実導入フェーズへ移行しつつある。
Google Gemma 4 によるローカルAI推論の実用化
Google がオープンソースモデル Gemma 4 をリリース。iPhone ネイティブ動作・完全オフライン推論への対応が注目を集め、「スマホに載せたローカルAIに未来を感じる」という反響が多数。エッジデバイスでの AI 実行が現実の選択肢に。
AIエージェント・マルチエージェント基盤の整備
Claude Code のサブエージェント活用ガイド、コーディングエージェント向けサンドボックス Freestyle の登場、AWS Frontier Agents による SRE 業務変革、fence によるOSサンドボックス実行など、エージェント実装の標準化が一週間で急速に進展した。
AI の社会・経済的影響への批判的議論の増加
ペンシルバニア大+ボストン大の論文が「AIリストラが経済を崩壊させる」を数理証明、スタンフォード報告書がAI関係者と一般市民の認識格差を指摘、「仕事の未来は嘘か」論考など、楽観論だけでない多面的な批判的検討が増加。
サプライチェーン・プライバシー攻撃の連発
CPU-Z / HWMonitor へのマルウェア混入、WordPress プラグイン31本へのバックドア埋め込み、FBI による削除済み Signal メッセージ復元、ロシア GRU による家庭用ルーター乗っ取り（1万8千台）など、1週間でサプライチェーン攻撃が連続発生。
数学・科学研究への AI 本格参入
「数学研究におけるAI革命が到来、数学者たちは『これはまだ始まりに過ぎない』と考えている」という記事が注目を集めた。Artemis II 帰還とも相まって、AI がサイエンスを加速する事例として広く共有された。
