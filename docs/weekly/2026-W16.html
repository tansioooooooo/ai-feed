📋 週次サマリー

2026年4月6〜12日の週は、AnthropicのClaude Mythosモデルが「ゼロデイ攻撃の自律開発」「サンドボックス脱出」を実証し一般公開が見送られたことが最大の話題となった。並行してClaude Code・Cowork・Managed Agentsの一般提供が加速し、日本企業での全社導入事例が相次いだ。GoogleはGemma 4をApache 2.0でオープンソースリリース。AIエージェント開発とバイブコーディングへの関心が急高まりする一方、AI依存による「認知的降伏」や採用AI差別訴訟など社会的懸念も浮上した週となった。

🔥 主要トレンド

Claude Mythosのセキュリティ能力と一般公開見送り
Anthropicの次世代モデル「Claude Mythos」がゼロデイ攻撃の自律開発やサンドボックス脱出能力を実証したとして、System Cardが公開されつつも一般提供は見送り。米財務長官らが緊急会合を開くほどの衝撃を与え、AI安全性規制の議論が急加速した。
Claude Code / Cowork / Managed Agentsの企業普及
AnthropicのClaude Cowork（自律作業エージェント）とManaged Agentsが一般提供を開始。日本では全社導入・1か月で200アプリ構築・PR自動レビュー83%達成などの事例が続出し、エンタープライズ向けAIコーディング市場が急成長している。
Google Gemma 4のオープンソースリリースとローカルAI競争
GoogleがGemma 4をApache 2.0ライセンスで公開。iPhone・Androidでのローカル動作やLM StudioとClaude Codeとの連携が即日確認され、オープンウェイトモデルの実用性が大幅に向上。1-bit LLM「Bonsai」（8B/1.15GB）も注目を集めた。
AIエージェント開発の本格化とセキュリティ課題
Claude Managed AgentsやFreestyle（Coding Agent Sandbox）など新ツールが相次ぎ登場。一方でエージェントが機密データを漏洩する5つの実例や、Workdayの採用AIが11億件の応募を差別的に却下したとして訴訟になる事例も報告され、権限管理の標準化が急務に。
バイブコーディング論争の深化
「バイブコーディングはドッグフーディング崇拝の暴走だ」（HN）という批判記事が注目を集める一方、日本では「エンジニア歴20年がAIコーディング初心者に物申す」「縦スクロールシューティングをバイブコーディングで世界初制作」など実践報告が続出。熟練者とAI初心者の対話が活発化。
AI社会リスクへの多面的な警鐘
「AIの誤情報を疑わず受け入れる認知的降伏が1372名の実験で確認」「AIによる支援が学習の粘り強さを低下させ成績を悪化させる」「AIは暴力的反発に遭う」など、AI依存の社会的・認知的コストに関する研究・論考が集中して掲載された。
