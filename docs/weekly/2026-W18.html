📋 週次サマリー

2026年4月20〜26日の1週間は、Claude Opus 4.7・DeepSeek V4・Kimi K2.6・Qwen3.6・GPT-5.5など主要AIモデルの新バージョンが相次いでリリースされ、性能競争が激化した。ハーネスエンジニアリングとAIエージェント開発が新たな注目テーマとして浮上し、Claude Code関連の実践記事が急増。一方、MetaやAtlassianのデフォルトデータ収集、MCPの深刻な脆弱性、複数企業の情報漏えいなどAI時代のプライバシー・セキュリティリスクも顕在化。日本政府がガバメントAI「源内」をオープンソース公開するなど、公共分野への展開も加速した。

🔥 主要トレンド

AIモデルの群雄割拠と性能競争の激化
Claude Opus 4.7、DeepSeek V4、Kimi K2.6、Qwen3.6-27B、GPT-5.5など、1週間で複数の新世代モデルが登場。オープンソースモデルがフラッグシップ級の性能を達成し、ローカル実行の現実性が高まった。また Google が Anthropic へ最大400億ドル（約6.4兆円）の投資を計画するなど、資本面でも競争が激化している。
ハーネスエンジニアリングの独立領域としての確立
AIエージェントを制御・管理する「ハーネスエンジニアリング」が独立した技術領域として注目を集め、実践的な記事・登壇が連日登場した。TAKT、MicrosoftのAPM、Claude Code Skillsなどのツールを活用したエージェント制御手法が盛んに議論されており、従来の開発規律をエージェントが扱えるよう再設計する流れが鮮明になった。
Claude Codeの現場普及と課題の露呈
Claude Codeの実践ガイド・セキュリティ勉強会・プラグイン紹介・Obsidian連携など、開発現場での採用が一気に広まった。同時に品質低下（1か月にわたるバグと設定変更が原因と判明）、Maxプラン限定化のテスト、Opus 4.7でのトークン消費量倍増といった課題も相次いで報告され、コミュニティの評価は二極化した。
AIと雇用・社会影響への本格論争
Anthropicの調査「プログラマーはAIに職を奪われる」、PdMの仕事の半分がAIに代替された体験談、「AI配転」で事務職を営業へ転換する動き、新卒採用縮小による第二の就職氷河期懸念など、AIが雇用・社会構造に与える具体的な影響についての議論が活発化した。羽生善治九段の「AI時代の差別化」発言も注目を集めた。
プライバシー・セキュリティリスクの同時多発
MetaとAtlassianによるデフォルトでの従業員・ユーザーデータ収集、MCPの深刻な脆弱性（関連ソフト1.5億DL超）、CAMPFIREの22万人超の個人情報漏えい、Bitwardenのサプライチェーン攻撃、Vercelの侵害など、AI時代特有のセキュリティリスクが一週間に集中して顕在化した。
公共・政府分野へのAI浸透
日本デジタル庁がガバメントAI「源内」をオープンソース公開（商用利用可）、自民党が1兆円規模のAI投資を要請、Google AI Pro認定講座の無料提供など、AI活用が公共セクターへ本格展開。一方でEUの年齢確認アプリがハッカーに2分で破られるなど、制度設計の課題も浮き彫りになった。
