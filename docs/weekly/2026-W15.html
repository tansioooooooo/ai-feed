📋 週次サマリー

2026年3月30日〜4月5日の1週間、AIコーディングツール（特にClaude Code）を巡る話題が集中した。ソースコードリーク疑惑、使用量制限問題、OpenClaw制限、Linuxの23年隠れていた脆弱性の発見など、Claude Codeが多角的に注目された。一方でGoogleのGemma 4リリース、OpenAIの評価額$852B達成、Qwen3.6など新モデルも相次いだ。セキュリティ面ではaxios NPMサプライチェーン攻撃が週をまたいで大きく報じられた。Artemis IIの月面ミッション成功も週の明るいニュースとなった。

🔥 主要トレンド

Claude Code の急速な普及と摩擦
Claude Codeのソースコードリーク騒動、ユーザーの使用量制限問題、OpenClaw接続の制限措置、さらにClaude Codeが23年間潜伏していたLinux脆弱性を発見したニュースまで、Anthropicのコーディングエージェントが週内で最も多く言及された。急激な普及に伴うインフラ・ポリシー面での課題が表面化した週だった。
AIコーディングツールの収益化論争
MicrosoftのGitHub CopilotがPRに広告を挿入していた問題が発覚し、150万件超のPRに影響したとして炎上。バックラッシュを受けてMicrosoftは数日で撤回した。AIツールの収益化がユーザー信頼と衝突するケースとして業界に警鐘を鳴らした。
新世代AIモデルの相次ぐリリース
GoogleがGemma 4オープンモデルをリリース、Qwen3.6-Plusがリアルワールドエージェントに向けた機能を発表、OpenAIが$852Bのバリュエーションで資金調達を完了、1-Bit LLMの商業化事例も登場するなど、モデル開発競争が加速した。
NPMサプライチェーン攻撃（axios）
広く使われるaxiosライブラリのNPMパッケージが改ざんされ、リモートアクセストロイが含まれた悪意あるバージョンが配布された。週をまたいで複数日でフォローアップ記事が上がり続け、オープンソースのサプライチェーンセキュリティへの関心を再燃させた。
AIとRAGの次世代アーキテクチャ
「RAGを仮想ファイルシステムに置き換えた」事例が注目され、コーディングエージェントの構成要素を解説する記事、セルフディスティレーションによるコード生成改善など、AIエージェントとデータ取得のアーキテクチャ刷新に関するコンテンツが集中した。
AIポリシー・規制の動向
EUがAIによるヌード加工アプリの全面禁止に合意、AI顔認識による誤逮捕事件（テネシー州）、日本でのAIバス赤字廃止事例、OkCupidが顔認識企業に300万枚の写真を提供していた問題など、AIの社会的影響と規制強化の動きが各国で報告された。
